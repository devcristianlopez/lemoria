# PRD — Lemoria

## Sistema Operativo de Memoria y Orquestación para Desarrollo con IA

### Problema

Las herramientas actuales de IA para programación presentan: pérdida de contexto entre sesiones, conversaciones desconectadas, decisiones técnicas no registradas, commits sin relación con requerimientos, documentación desactualizada, agentes sin memoria persistente.

### Solución

Lemoria es una plataforma local-first que proporciona:
- Memoria persistente y estructurada
- Coordinación multiagente
- Trazabilidad técnica completa
- Integración con GitHub
- Sincronización con Obsidian
- Desarrollo guiado por especificaciones (SDD)

### MVP Fase 1

- PostgreSQL como núcleo persistente
- Agente mayor/orquestador
- Sistema de proyectos
- Memoria estructurada
- Manejo de PRDs
- Sistema de tareas
- Registro de decisiones
- Historial de conversaciones
- Integración básica con GitHub
- Sincronización Markdown con Obsidian
- Flujo básico SDD

### Métricas de éxito

- Reducción de pérdida de contexto
- Continuidad entre sesiones
- Trazabilidad completa
- Reutilización de conocimiento
- Reducción de prompts repetitivos
