Guardian Live Monitor MVP Demo. Plataforma de monitoreo de seguridad que captura video local (OpenCV), detecta movimiento por frame differencing, envía eventos vía HTTP POST a una API FastAPI que los persiste en PostgreSQL y los publica en Redis Pub/Sub. Un worker interno reenvía los eventos por WebSocket a un dashboard web (Vanilla JS + Tailwind CSS) que muestra un feed en vivo con analítica básica. Despliegue con Docker Compose.
