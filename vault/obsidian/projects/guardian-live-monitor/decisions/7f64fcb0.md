# ADR: Implementar detector de movimiento por visión computacional

**Status**: proposed
**ID**: 7f64fcb0
**Proyecto**: [[projects/guardian-live-monitor/README|guardian-live-monitor]]

## Description

Se creó detector/detector.py con frame differencing (absdiff + threshold + dilate). Config vía env vars. Envío asíncrono de eventos a POST /api/events con httpx. Cooldown y severidad dinámica según confidence.

## Rationale

Usar background subtraction (MOG2/KNN) -> se descartó por mayor complejidad y consumo. Usar deep learning -> overkill para detección binaria de movimiento. Frame differencing simple elegido por simplicidad, bajo costo CPU y suficiencia para el caso de uso.
