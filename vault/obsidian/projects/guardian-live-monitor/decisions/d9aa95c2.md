# ADR: M1.2: Dockerization and infrastructure wiring

**Status**: proposed
**ID**: d9aa95c2
**Proyecto**: [[projects/guardian-live-monitor/README|guardian-live-monitor]]

## Description

Created Dockerfile for Python 3.12-slim image exposing port 8000 with uvicorn. The docker-compose.yml (pre-existing) already wires postgres, redis, backend, detector, and dashboard services together.

## Rationale

Alternatives considered: python:3.12 (slim preferred for smaller image), gunicorn + uvicorn workers (not needed for single-process async app).
