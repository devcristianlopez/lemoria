# ADR: Algoritmo de detección de movimiento

**Status**: proposed
**ID**: e86522d8
**Proyecto**: [[projects/guardian-live-monitor/README|guardian-live-monitor]]

## Description

Se implementó frame differencing simple (absdiff + threshold + dilate) por su eficiencia computacional y simplicidad. El detector genera video sintético si no encuentra archivo de prueba, permitiendo demo sin webcam. Cooldown de 3s entre eventos para evitar saturación. Envío asíncrono con httpx. Severidad dinámica basada en confidence (proporción de píxeles en movimiento).

## Rationale

Alternativas: MOG2 (mejor en condiciones de luz variable pero más pesado), KNN (similar), deep learning (overkill). Frame differencing es suficiente para el caso de uso del MVP con fondo estático.
