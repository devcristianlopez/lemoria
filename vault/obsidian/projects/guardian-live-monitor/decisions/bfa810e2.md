# ADR: Variable global shadoweada por asignación local en main()

**Status**: proposed
**ID**: bfa810e2
**Proyecto**: [[projects/guardian-live-monitor/README|guardian-live-monitor]]

## Description

En detector.py, la variable _latest_jpeg se asigna dentro de main() sin declaración 'global'. Python trata la asignación como creación de variable local, no como modificación de la global. El MJPEG Handler (en otra función) lee la global que nunca cambia. Fix: agregar 'global _latest_jpeg' en main().

## Rationale

Alternativas consideradas: usar dict/list mutable compartido (más complejo), pasar _latest_jpeg por referencia a main() (no aplica)
