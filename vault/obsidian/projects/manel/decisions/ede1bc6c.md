# ADR: Tech Stack: Electron + React + TypeScript + Tailwind + SQLite

**Status**: proposed
**ID**: ede1bc6c
**Proyecto**: [[projects/manel/README|manel]]

## Description

Stack definido en PRD MVP: Frontend con Electron 28+, React 18+, TypeScript 5+, Tailwind 3+. Backend local Node.js con better-sqlite3 para base SQLite. Build tool: electron-vite o Vite + electron-builder. Alternativas consideradas: Tauri (Rust) descartado por simplicidad MVP, Python + Qt descartado por experiencia UI moderna.
