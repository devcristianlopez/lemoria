Manel — Security Health Monitor para entornos de desarrollo. MVP: scanner local que detecta software instalado (SO, lenguajes, herramientas), consulta vulnerabilidades (NVD, CVE, OSV, GitHub Advisories), calcula Security Score (0-100), muestra dashboard Electron con React+TypeScript+Tailwind. Base SQLite local. Ver PRD completo proporcionado por el usuario (versión 1.0, Draft MVP, Junio 2026).
