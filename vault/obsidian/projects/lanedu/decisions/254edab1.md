# ADR: Stack tecnológico Lanedu: HTML5 + TailwindCSS + GitHub Pages

**Status**: proposed
**ID**: 254edab1
**Proyecto**: [[projects/lanedu/README|lanedu]]

## Description

Se eligió HTML5 + TailwindCSS (CDN) + Vanilla JS por simplicidad y compatibilidad con GitHub Pages. FormSubmit como solución de formulario gratuita sin backend. Paleta azul corporativo + blanco/gris para transmitir confianza profesional.

## Rationale

Alternativas consideradas: Next.js/React — overkill para una landing page estática. Netlify Forms — buena alternativa pero requiere cuenta externa. FormSubmit se eligió por ser gratuito, no requerir backend y ser compatible con GitHub Pages.
