# ADR: 7 reusable skills created

**Status**: proposed
**ID**: c971366f
**Proyecto**: [[projects/lemoria/README|lemoria]]

## Description

Skills for frontend, backend, database, testing, code-review, git-workflow, documentation created globally at ~/.config/opencode/skills/. Each skill has a SKILL.md with use-cases, best practices, and examples. Skills are loaded manually by the agent (not auto-loaded).

## Rationale

Put all knowledge in agent prompts, Use a single monolith skill
