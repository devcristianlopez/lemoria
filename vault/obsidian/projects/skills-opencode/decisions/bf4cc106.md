# ADR: README y publicación en GitHub

**Status**: proposed
**ID**: bf4cc106
**Proyecto**: [[projects/skills-opencode/README|skills-opencode]]

## Description

Se creó README.md con documentación completa de los skills, instrucciones de instalación (3 métodos), estructura del repo, y guía de desarrollo. Repo subido a github.com/devcristianlopez/skills-opencode (público) con commit inicial de 4 archivos y 789 líneas.

## Rationale

Alternativas: (1) README mínimo solo con enlaces (descartado, poco útil). (2) README + wiki separada (descartado, sobreingeniería para el tamaño del proyecto).
